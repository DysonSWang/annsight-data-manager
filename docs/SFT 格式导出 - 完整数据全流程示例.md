## 前端界面操作

### 任务列表页导出

**路径**: `/finetuning/list.html`

1. 打开微调任务列表页
2. 找到已完成审核的任务
3. 点击任务卡片上的 **"导出 SFT 数据"** 按钮
4. 自动下载 JSONL 文件

### 任务详情页导出

**路径**: `/finetuning/task-detail.html?id=ft-7b4eef3a-fc34-4905-a423-e4a80502d490`

1. 点击任务列表中的 "查看详情" 进入任务详情页
2. 点击右上角的 **"导出 SFT 数据"** 按钮
3. 在弹出的对话框中选择：
   - **导出格式**: SFT 格式（带 <think> 思考标签）
   - **数据范围**: 全部已审核通过的数据
4. 点击 "导出" 按钮
5. 浏览器自动下载文件：`sft-export-{taskId}-{timestamp}.jsonl`
