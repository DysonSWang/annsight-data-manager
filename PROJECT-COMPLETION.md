## 🔧 技术实现

### 架构设计
```
┌─────────────────────────────────────────────────────────┐
│                     前端页面层                            │
│  (index.html | raw-data.html | stats.html | spotcheck.html)│
└─────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────┐
│                     Express.js 路由层                     │
│  (/api/raw-data | /api/review | /api/etl)               │
└─────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────┐
│                   Repository 数据访问层                   │
│  (RawDataIndexRepository | ProcessedDataRepository)     │
└─────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────┐
│                     PostgreSQL 数据库                    │
└─────────────────────────────────────────────────────────┘
```

### 关键技术决策
- **数据库**: PostgreSQL (统一，避免 SQLite 兼容性问题)
- **异步处理**: 队列模式（ETL 异步处理）
- **去重机制**: MD5 完全重复检测（MinHash 语义去重待实现）
- **置信度阈值**: 0.8（可配置，支持动态调整）
- **冷却期**: 24 小时（AI 自动通过数据）
